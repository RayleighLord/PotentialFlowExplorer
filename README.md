# Potential Flow Explorer

An interactive browser-based explorer for classical two-dimensional potential flows in incompressible, irrotational aerodynamics.

The app uses analytical elementary solutions — uniform flow, sources, sinks, vortices, and doublets — and visualizes their superposition with:

- a full-window dark canvas,
- animated tracer particles,
- seeded streamlines,
- movable singularities,
- preset examples such as flow past a cylinder and image-system wall flows.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
